🌟 GhibliSwiftUI

Explore Studio Ghibli movies in a sleek iOS SwiftUI app! 🎬✨ Smooth animations, beautiful movie details, and easy navigation bring the Ghibli universe to your device.

🎥 Features

Browse a list of Studio Ghibli movies with posters.

Tap for movie details: description, director, release date.

Smooth animations & transitions: fade, scaling, parallax effects.

MVVM architecture & async API fetching.

Search movies by title.

Works on iOS 16+ with SwiftUI.
